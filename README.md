# header-42 package

An atom package to add the 42 header to your files.

You can trigger it with the `cmd-alt-h` shortcut.

supported header formats are:
`#`, `/* */`, `<!-- -->`, `[[ ]]`, `(* *)`, `{- -}`, `;`

[Image is from nasa APOD website](http://apod.nasa.gov/apod/ap150731.html)
