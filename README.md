# header-42 package

An atom package to add the 42 header to your files.

You can trigger it with the `cmd-alt-h` shortcut.

![42 header in action](http://i.imgur.com/xyGCPjm.png)

### supported file extensions:
- `#`
	- Makefile
- `/* */`
	- c
	- cpp
	- h
	- js
	- css
	- cs
	- scala
	- rs
	- go
	- swift
	- php
- `<!-- -->`
	- html
- `[[ ]]`
	- lua
- `(* *)`
	- ml
	- mli
- `{- -}`
	- hs
- `;`
	- s
	- s64
	- asm
	- hs
	- h64
	- inc
